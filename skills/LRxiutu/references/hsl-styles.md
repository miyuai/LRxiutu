# HSL 肤色双风格

**Orange Hue 是方向舵**——向黄 = 暖金，向红 = 白红透亮。

## 🅰️ 暖金肤（默认，黄金时刻/户外暖光）

| 通道 | Hue | Saturation | Luminance |
|------|-----|-----------|-----------|
| 🟠 Orange | **+8** (向黄) | -12 | +22 (黄金→+25) |
| 🔴 Red | +8 (向橙) | +5 | -5 |
| 🟡 Yellow | +8 (向橙) | -15 (黄金→-10) | — |

联动：Temperature=时段值, Tint=+8, Vibrance=+10, Dehaze=-8, Clarity=-8, CG高光Sat=30

## 🅱️ 白红透亮（室内/冷光/亚洲审美）

| 通道 | Hue | Saturation | Luminance |
|------|-----|-----------|-----------|
| 🟠 Orange | **-8** (向红) | **-18** | **+28** |
| 🔴 Red | **0** (自然) | **+10** | **+5** |
| 🟡 Yellow | 0 | **-20** | — |

联动：Temperature=时段值**-150K**, Tint=**+12**, Vibrance=**+5**, Dehaze=**-10**, Clarity=**-10**, CG高光Sat=**15**

> Tint +12 是关键：品红底色对冲草绿反光，白里透红的底盘。

## 通用通道（两风格共用）

| 通道 | Saturation |
|------|-----------|
| 🟢 Green | -15 (白红→-20) |
| 🟣 Purple | -10 |
| 🟣 Magenta | -8 |

## 时段无关固定参数

| 参数 | 值 |
|------|-----|
| Contrast2012 | -5 (清透→-2) |
| Saturation | -5 |
| Texture | +5 |
| SharpenEdgeMasking | 40 |
| ToneCurvePV2012 | [0,12, 32,34, 64,68, 128,130, 192,195, 224,228, 255,248] |
